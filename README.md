# Claude Trader 🤖📈

An adaptive AI portfolio experiment where Claude manages a $1,000 paper trading portfolio,
learns from every trade, and improves its strategy over time.

---

## Quick Setup

### 1. Clone & install
```bash
git clone <your-repo>
cd claude-trader
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Get your API keys

| Key | Where to get it | Cost |
|-----|----------------|------|
| `ALPACA_KEY` + `ALPACA_SECRET` | alpaca.markets → Paper Trading | Free |
| `ANTHROPIC_KEY` | console.anthropic.com | Pay per use (~$0.05/cycle) |
| `TAVILY_KEY` | tavily.com | Free tier available |

### 3. Configure
```bash
cp .env.example .env
# Edit .env and paste your keys
```

### 4. First run (dry run — no trades placed)
```bash
python main.py --dry-run
```

### 5. Check portfolio status only
```bash
python main.py --status
```

### 6. Live run (executes real paper trades)
```bash
python main.py
```

## Automation (every 3 trading days)

### Option A: Cron job (Linux/Mac)
```bash
crontab -e
# Add this line (runs at 10am on Mon/Wed/Fri):
0 10 * * 1,3,5 cd /path/to/claude-trader && /path/to/.venv/bin/python main.py >> logs/cron.log 2>&1
```

### Option B: Windows Task Scheduler
- Open Task Scheduler → Create Basic Task
- Trigger: Weekly, Mon/Wed/Fri at 10:00 AM
- Action: Start a program → `python` → Arguments: `main.py` → Start in: `C:\path\to\claude-trader`

### Option C: VPS (fully hands-off)
```bash
# On a $5/month DigitalOcean droplet:
crontab -e
0 10 * * 1,3,5 cd ~/claude-trader && python main.py >> logs/cron.log 2>&1
```

## How it learns

Every 3 trading days:
1. Claude reads its full trade history and lessons
2. Claude researches the topic it requested last cycle
3. Claude reflects: "what worked, what didn't?"
4. Claude makes new decisions based on updated thinking
5. Claude rewrites its own strategy notes
6. After every 15 trades → oldest 10 are compressed into a "lesson batch"

The `memory/brain.json` is the heart of the experiment.
Open it anytime to read Claude's evolving thoughts.
